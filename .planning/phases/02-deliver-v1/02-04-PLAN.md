# Phase 02 Plan 04: Refine quiz progression UX

## Objective

Restructure the public Easter quiz so visitors move through one question at a time, do not receive correctness feedback, and can optionally reveal the related Bible verse location as a hint.

## Context

- @.planning/PROJECT.md
- @.planning/ROADMAP.md
- @.planning/STATE.md

## Tasks

1. [x] Replace the all-questions-on-one-screen quiz flow with a sequential step-based interaction.
2. [x] Remove correctness and score feedback from the public quiz while preserving answer capture.
3. [x] Add a hint control that reveals the current question's related Bible verse reference without exposing the answer.
4. [x] Refresh the quiz layout and controls so the new flow reads clearly on desktop and mobile.

## Verification

- `npm run lint` should pass.

## Success Criteria

- `/kviz` shows only one question at a time.

- The public quiz no longer reveals whether answers were correct.
- A hint button reveals the related Bible verse location for the active question.
